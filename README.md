# えいご まいにちレッスン

小学3年生がひとりで学べる、親子の日常会話をテーマにした毎日1レッスン（約15分）の英会話学習アプリです。

## レッスンの流れ

1. きのうのおさらい
2. きく（リスニング）
3. はなす（スピーキング・録音して聞き比べ）
4. たんご
5. フレーズ
6. おまけ：フォニックス

クリアするとエサがもらえて、生き物にあげると育っていきます。1匹育て終えたら新しいたまごを選んで次の生き物を育てられます。

## 技術構成

- 単一 `index.html`（HTML / CSS / JavaScript のみ、外部ライブラリなし）
- 音声：ブラウザ内蔵の Web Speech API（`en-GB` を優先使用）
- 録音：MediaRecorder API（保存はせず、その場で聞き比べのみ）
- データ保存：localStorage

## 使い方

`index.html` をブラウザで開くだけで動作します。iPad / iPhone の Safari を推奨します。
